What is Side Scroller 3D Kit?

The Side Scroller 3D Kit is an easy to use template for creating side scroller type of games in 3D! Supports both desktop and mobile platforms The template is created using Blueprints, is fully documented and allows easy customization.

Player Character

The player character has the ability to jump, attack, and roll.

Monsters of 5 Type

Melee monsters, ranged monsters, elite monsters as well as bosses are also provided.

Pickup Items and Traps

This package has pickup items(HP, Gold) and traps of 2 type
